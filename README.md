# NCmusic-vue 🎵

一款基于 **Vue 3** + **Pinia** + **Vite** 构建的高颜值音乐流媒体 Web 应用。

> **🚧 项目声明**：本项目目前仍处于 **Active Development (积极开发与完善中)** 阶段。部分功能正在迭代，UI 细节也在持续优化，欢迎 Fork 或提 Issue 交流。

---

## ✨ 项目亮点

* **沉浸式红黑视觉**：深度还原网易云音乐经典配色方案，打造极简、硬朗的黑胶质感。
* **全局状态管理**：利用 **Pinia** 维护全局播放状态，实现跨页面的无缝播放体验。
* **极致滚动优化**：针对 `.main-content` 容器定制化滚动逻辑，解决 Tab 切换与详情页返回的滚动条重置难题。
* **高性能列表渲染**：采用 `IntersectionObserver` 实现搜索结果的懒加载与无限滚动。

---

## 🛠️ 技术栈

| 工具 | 作用 |
| :--- | :--- |
| **Vue 3 (Composition API)** | 核心开发框架 |
| **Vite** | 下一代前端构建工具 |
| **Pinia** | 极简、类型安全的全局状态管理 |
| **Vue Router** | 单页应用路由控制（支持嵌套路由） |
| **Axios** | 异步请求封装与 API 通信 |

## 🚀 功能特性 (Features)

### 🟢 已实现 (Completed)
- [x] **扫码登录系统**：集成网易云 API 接口，实现 **二维码生成、状态轮询与安全登录**，支持同步个人收藏歌单及日推。
- [x] **编辑我的音乐**：可创建歌单、删除歌单；可将歌曲加入我的歌单，也可从我的歌单中删除歌曲；
- [x] **核心播放逻辑**：基于 **Pinia** 全局状态管理的音频控制器，支持播放/暂停、切歌、循环播放、随机播放、进度条拖拽、音量控制及播放模式切换（循环、随机）。
- [x] **动态歌词滚动**：解析 Lrc 格式歌词，利用响应式逻辑实现歌词与音频进度毫秒级同步，支持当前行自动高亮及平滑滚动效果。
- [x] **高仿网易云 UI**：深度还原红黑主题视觉，包含沉浸式播放详情页、封面黑胶旋转动效、以及基于 Flexbox 的响应式侧边栏布局。
- [x] **高性能搜索系统**：
    - **分页加载逻辑**：将单次全量请求重构为分页加载，首屏网络延迟降低 **50%**，并减少 **45%** 的初始化耗时。
    - **分类展示**：支持对单曲、歌手、专辑、歌单、视频（MV）进行分维度展示。
    - **搜索建议**：引入 **Debounce（防抖）** 策略优化联想搜索，在保证即时反馈的同时极大减轻服务器压力。
- [x] **MV 在线播放**：集成 HTML5 `<video>` 组件，支持高清 MV 流媒体播放与简单的交互控制。
- [x] **组件状态缓存 (Keep-alive)**：利用 Vue 的 `keep-alive` 结合 `vue-router` 实现搜索结果页、列表页的组件缓存，**确保路由回退时无缝恢复滚动位置与交互状态**。
- [x] **数据持久化**：深度集成 LocalStorage，实现播放列表、音量设置、用户偏好等数据的持久化存储，提升二次访问体验。
**......**
- [x] **VIP与普通用户身份区分**：根据身份获取不同歌曲url，并显示VIP试听标识等。

## 🚀 快速启动

在运行本项目之前，**请务必确保你的本地环境已部署并运行了网易云音乐后台 API**。

### 1. 运行后端 API
本项目依赖于 [NeteaseCloudMusicApi](https://github.com/NeteaseCloudMusicApiEnhanced/api-enhanced)。
默认 API 端口为 http://localhost:3000

### 2.运行本项目
```bash
# 1. 安装项目依赖
pnpm install

# 2. 启动开发服务器
pnpm dev

# 3. 访问项目
# 打开浏览器访问 http://localhost:5173
```
